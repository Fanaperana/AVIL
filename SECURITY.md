# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | ✅        |

## Reporting a Vulnerability

If you discover a security issue in this repository (e.g., a compiled-PDF exploit,
leaked credentials in git history, or a supply-chain concern in the LaTeX packages),
please **do not open a public issue**.

Instead, email **fanaperanaprince@gmail.com** with:

1. A description of the vulnerability
2. Steps to reproduce (if applicable)
3. Any potential impact you've identified

You should receive an acknowledgment within **48 hours**. We will work with you to
understand and address the issue before any public disclosure.

## Scope

This policy covers:

- The LaTeX source files and build pipeline in this repository
- The compiled PDF artifact (`build/avil.pdf`)
- Any future code (reference implementations, evaluation scripts) added to this repo

Thank you for helping keep this project safe.
